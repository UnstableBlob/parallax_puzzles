# Paraallax Quest

A premium 5-game challenge built with Next.js and React.

## Features
- **Progression System**: Complete one game to unlock the next.
- **Glassmorphism UI**: High-end aesthetic with smooth transitions and premium gradients.
- **Responsive Design**: Works on all screen sizes.

## Getting Started
1. Install dependencies: `npm install`
2. Start the dev server: `npm run dev`
3. Open `http://localhost:3000`
